# Pick List Templates

This directory contains template reference images for the Pick List Scanner.

## Template Files

- **template-a.jpg** - Template A (Structured Pick List): Table/grid format with columns for product name, net weight, and quantity
- **template-b.jpg** - Template B (Visual/Large Text): Large text format with product images (coming soon)

## Adding Template Images

To add or update template images:

1. Save your template image to this directory
2. Use the filename format: `template-a.jpg` or `template-b.jpg`
3. Supported formats: JPG, PNG
4. Recommended: Keep file sizes reasonable for web display (under 2MB)

The Pick List Scanner component will automatically display these images in the Template Preview section.
